Organization

Get organization

The above command returns JSON structured like this:

{
"id": 1,
"name": "My organization",
"admin_id": 1
}

This endpoint information about the current organization

HTTP Request

GET /organization

Expand Parameters

Parameter Description
admin Organization admin user
users List of users
roles List of user roles
issueLists List of issue lists
ratings List of ratings
